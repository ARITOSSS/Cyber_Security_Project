File Upload Vulnerability

This page demonstrates a security misconfiguration vulnerability in file uploads. You can upload any type of file without restrictions, which might allow an attacker to upload malicious files.

Try uploading an html file with a script inside and put "http://127.0.0.1:8000/media/name_of_your_file" in the url.

This happen because this page don't check the type of file you upload.
